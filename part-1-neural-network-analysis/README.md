# Neural Network Fundamentals and Training Behavior Analysis

## Project Objective
The objective of this project is to build and analyze a neural network model for customer churn prediction using deep learning techniques.

## Dataset
Customer Churn Neural Network Dataset

## Tasks Performed
- Dataset understanding
- Data preprocessing
- Neural network model building
- Training and evaluation
- Hyperparameter experimentation
- Final reflection analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow/Keras

## Model Details
A feed-forward neural network was developed using TensorFlow/Keras with:
- Hidden layers
- ReLU activation
- Sigmoid output activation
- Adam optimizer

## Evaluation Metrics
- Accuracy
- Loss
- Confusion Matrix
- Classification Report

## Conclusion
The neural network model successfully learned patterns from the dataset and achieved good predictive performance for customer churn classification.
